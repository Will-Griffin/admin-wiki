## Identity and access management

Identity and access management covers multiple aspects such as  API authentication, data isolation, and provider access.

## Design considerations

- Isolate department access to patient data.
- Providers should be granted explicit access to a department's patient data.
- A single provider application registration should be used to represent the external system.
