## Overview

This page lists recommended training resources for Development Team members. New team members should complete the **Onboarding Track** within their first 90 days. All team members are expected to complete at least two courses per quarter to stay current with the evolving technology stack.

## Recommended Learning Paths

### For Junior Developers
1. Complete C# Masterclass (Udemy)
2. Git: Getting Started (Pluralsight)
3. REST API Design (Udemy)
4. AZ-204 Azure Developer (Udemy)
5. Azure DevOps: Getting Started (Pluralsight)
6. Unit Testing with xUnit and Moq (Pluralsight)

### For Mid-Level / Senior Developers
1. C# Advanced Topics (Udemy)
2. Azure Functions Deep Dive (Pluralsight)
3. Design Patterns in C# and .NET (Udemy)
4. Azure APIM (Udemy)
5. HL7 FHIR for Developers (Udemy)
6. .NET Microservices (Udemy)
7. Azure Solutions Architect AZ-305 (Udemy)

### For Tech Leads / Architects
1. Software Architecture & Design Patterns (Udemy)
2. Domain-Driven Design Fundamentals (Pluralsight)
3. Azure Solutions Architect AZ-305 (Udemy)
4. Microservices Architecture (Udemy)
5. Azure Key Vault & App Configuration (Pluralsight)
