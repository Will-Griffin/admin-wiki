## Integration Services Developer I (Junior Developer)

### Summary
Entry-level software developer responsible for implementing features and bug fixes on Azure-based and BizTalk integration solutions under the guidance of senior developers.

### Key Responsibilities
- Develop and maintain Azure Functions, Logic Apps, and APIM policies for integration workflows.
- Write unit and integration tests for all code changes.
- Participate in code reviews as a reviewer and author.
- Contribute to CI/CD pipeline improvements in Azure DevOps.
- Document technical work in wiki and code comments.
- Participate in daily standups, sprint planning, and retrospectives.
- Troubleshoot issues escalated from the Support Team.

### Required Skills & Qualifications
- Bachelor's degree in Computer Science, Software Engineering, or related field.
- Proficiency in C# / .NET (6.0+).
- Understanding of REST API principles and HTTP.
- Familiarity with Git source control.
- Basic knowledge of Azure services (Functions, Service Bus, APIM, Storage).
- Ability to write and understand unit tests.

### Preferred Skills
- Experience with Azure DevOps (boards, repos, pipelines).
- Exposure to EDI standards (X12) or healthcare data (HL7, FHIR).
- Familiarity with BizTalk Server.
- Microsoft Azure Fundamentals (AZ-900) or Azure Developer (AZ-204) certification.

### Tools Used
C#, .NET, Azure Functions, Azure APIM, Azure DevOps, Git, Visual Studio / VS Code, Postman, Azure Portal

## Integration Services Developer II (Mid-Level Developer)

### Summary
Mid-level developer responsible for independently designing and implementing integration solutions, leading code reviews, and mentoring junior developers. Key contributor to architecture discussions and process improvements.

### Key Responsibilities
- Design and implement complex Azure integration workflows (Functions, APIM, Service Bus, Logic Apps).
- Develop and maintain BizTalk Server maps, orchestrations, and pipelines.
- Lead feature development from design through deployment.
- Author and review architectural design documents.
- Write comprehensive unit, integration, and API tests.
- Mentor junior developers and conduct code reviews.
- Participate in incident response and root cause analysis.
- Contribute to CI/CD pipeline improvements and deployment automation.
- Collaborate with Support and DevOps teams on operational issues.

### Required Skills & Qualifications
- 3+ years of software development experience.
- Strong proficiency in C# / .NET with async programming.
- Hands-on experience with Azure Functions, Azure Service Bus, Azure APIM.
- Experience building and consuming REST APIs with OpenAPI specifications.
- Understanding of SOLID principles and common design patterns.
- Experience with Azure DevOps pipelines (YAML).
- Familiarity with FHIR, HL7, or EDI X12 standards.

### Preferred Skills
- BizTalk Server development (maps, orchestrations, EDI adapters).
- Azure Key Vault, App Configuration, Managed Identity patterns.
- Experience with Azure Monitor, Application Insights instrumentation.
- Microsoft AZ-204 (Azure Developer Associate) certification.
- Container experience (Docker, AKS).

### Tools Used
C#, .NET, Azure Functions, Azure APIM, Azure Service Bus, BizTalk Server, Azure DevOps, Git, GitHub, Visual Studio, Application Insights, Postman, Bruno

## Integration Services Senior Developer / Tech Lead

### Summary
Senior-level developer responsible for technical leadership of the Development Team, including architecture decisions, mentoring, cross-team coordination, and driving the evolution of the integration platform.

### Key Responsibilities
- Lead technical design and architecture for new integration solutions.
- Define and enforce coding standards, patterns, and best practices.
- Own the technical roadmap for Azure and BizTalk modernization.
- Lead performance optimization and system reliability improvements.
- Partner with the DevOps Team on infrastructure-as-code and pipeline design.
- Represent the Development Team in cross-functional planning and architecture reviews.
- Drive adoption of new technologies and platforms.
- Manage technical debt backlog; prioritize modernization efforts.
- Conduct technical interviews for new team members.
- Serve as final escalation point for complex technical issues.

### Required Skills & Qualifications
- 6+ years of software development experience, with at least 2 in a senior/lead role.
- Expert-level C# / .NET skills.
- Deep expertise in Azure Integration Services (APIM, Functions, Service Bus, Logic Apps).
- Experience with BizTalk Server 2016/2020 development and administration.
- Strong knowledge of FHIR R4, HL7, EDI X12 standards.
- Proven experience designing scalable, secure cloud-native architectures.
- Experience with Infrastructure-as-Code (Bicep or Terraform).
- Strong understanding of OAuth2, OpenID Connect, Azure AD/Entra.
- Microsoft AZ-204 or AZ-305 (Solutions Architect) certification required.

### Preferred Skills
- Azure DevOps Engineer Expert (AZ-400).
- Experience with healthcare regulatory requirements (HIPAA, CMS).
- Knowledge of Domain-Driven Design (DDD) and event-driven architecture.
- Experience with API governance and developer portal management.

### Tools Used
All tools used by the Development Team, plus: Bicep/Terraform, Azure Architecture diagrams, ADO reporting, Power BI, Architecture Decision Records (ADR)
