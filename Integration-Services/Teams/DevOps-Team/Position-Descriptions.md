## Integration Services DevOps Engineer I (Junior)

### Summary
Entry-level DevOps engineer responsible for supporting CI/CD pipeline operations, basic infrastructure provisioning, and monitoring tasks under the direction of senior engineers or the DevOps Lead.

### Key Responsibilities
- Monitor CI/CD pipeline runs in Azure DevOps; investigate and triage failures.
- Assist with Azure resource provisioning using IaC templates (Bicep/Terraform).
- Maintain and update YAML pipeline configurations under guidance.
- Monitor Azure dashboards (Application Insights, Log Analytics, Azure Monitor).
- Assist with access provisioning requests (Azure RBAC, Azure DevOps).
- Write and maintain runbooks and operations documentation.
- Participate in daily standups and change control processes.
- Support incident response under the direction of senior engineers.

### Required Skills & Qualifications
- Bachelor's degree in Computer Science, Information Technology, or related field.
- Familiarity with CI/CD concepts and tools (Azure DevOps or GitHub Actions).
- Basic understanding of cloud infrastructure (Azure preferred).
- Basic scripting skills (PowerShell or Bash).
- Understanding of Git source control.
- Strong written communication skills.

### Preferred Skills
- Microsoft Azure Fundamentals (AZ-900) certification.
- Hands-on experience with Azure Portal.
- Exposure to Infrastructure-as-Code (Bicep, Terraform, ARM).
- Familiarity with Docker or container concepts.
- ITIL Foundation certification or coursework.

### Tools Used
Azure Portal, Azure DevOps, GitHub Actions, Azure Monitor, Log Analytics, Application Insights, PowerShell, Git, Bicep/Terraform (learning)

## Integration Services DevOps Engineer II (Mid-Level)

### Summary
Mid-level DevOps engineer responsible for designing and maintaining CI/CD pipelines, provisioning Azure infrastructure via IaC, managing observability tooling, and enforcing security and RBAC standards.

### Key Responsibilities
- Design and maintain YAML pipelines in Azure DevOps and GitHub Actions.
- Provision and manage Azure resources via Bicep or Terraform.
- Configure and tune Azure Monitor alerts, dashboards, and workbooks.
- Manage Azure DevOps environment approvals, service connections, and RBAC.
- Implement and maintain secret management via Azure Key Vault.
- Perform quarterly access reviews for Azure and Azure DevOps.
- Manage certificate lifecycle (expiry tracking, rotation).
- Coordinate production deployments through the Change Control process.
- Assist Development Team with pipeline troubleshooting and optimization.
- Mentor DevOps Engineer I staff.

### Required Skills & Qualifications
- 3+ years of DevOps or cloud engineering experience.
- Hands-on experience with Azure DevOps pipelines (YAML).
- Proficiency in PowerShell and/or Bash scripting.
- Experience provisioning Azure resources via Bicep or Terraform.
- Strong understanding of Azure RBAC, Managed Identity, and Azure AD.
- Experience with Azure Monitor, Log Analytics, and Application Insights.
- Familiarity with Docker and container technologies.
- Microsoft Azure Administrator (AZ-104) certification required.

### Preferred Skills
- Azure DevOps Engineer Expert (AZ-400) certification.
- Experience with Azure Kubernetes Service (AKS).
- Knowledge of GitHub Actions and GitHub Environments.
- Experience with HashiCorp Vault or Azure Key Vault automation.
- ITIL 4 Foundation certification.

### Tools Used
Azure Portal, Azure DevOps, GitHub Actions, Bicep, Terraform, PowerShell, Azure Monitor, Log Analytics, Application Insights, Azure Key Vault, Docker

## Integration Services DevOps Lead / Senior Engineer

### Summary
Senior-level engineer responsible for leading the DevOps Team's operations, architecture decisions for cloud infrastructure, platform reliability, security posture, and driving the integration platform's DevOps maturity.

### Key Responsibilities
- Lead design and implementation of the CI/CD platform and deployment architecture.
- Own the Infrastructure-as-Code strategy and repository standards.
- Establish and enforce DevOps standards, pipeline templates, and best practices.
- Lead the monitoring, alerting, and observability strategy.
- Own the security posture for Azure environments: RBAC, Key Vault, Managed Identity.
- Lead quarterly access reviews and security audits.
- Manage the release management process in collaboration with Development and Support leads.
- Represent the DevOps Team in architecture and cross-functional planning discussions.
- Drive adoption of SRE (Site Reliability Engineering) practices.
- Manage the DevOps Team workload and capacity.
- Interview and onboard new DevOps Team members.

### Required Skills & Qualifications
- 6+ years of DevOps, cloud engineering, or platform engineering experience.
- Expert knowledge of Azure DevOps and GitHub Actions (pipeline authoring and governance).
- Deep expertise in Azure Infrastructure (AZ-104 / AZ-305 level).
- Strong IaC experience with Bicep and/or Terraform at scale.
- Expert-level PowerShell and scripting automation.
- Deep knowledge of Azure security: AAD/Entra ID, RBAC, Key Vault, Managed Identity, Defender for Cloud.
- Experience with container orchestration (Docker, AKS).
- Microsoft AZ-400 (DevOps Engineer Expert) certification required.
- Microsoft AZ-104 (Administrator) required; AZ-305 (Architect) preferred.

### Preferred Skills
- Azure Security Technologies (AZ-500) certification.
- Azure Solutions Architect Expert (AZ-305) certification.
- Site Reliability Engineering (SRE) experience.
- Experience with chaos engineering or failure injection testing.
- Healthcare cloud compliance experience (HIPAA, FedRAMP).
- HashiCorp Terraform Associate certification.

### Tools Used
All DevOps tools, plus: Azure Policy, Azure Defender/Sentinel, Terraform Cloud, Power BI (ops reporting), architecture documentation tools
