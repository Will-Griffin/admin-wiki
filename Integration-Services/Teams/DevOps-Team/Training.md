## Overview

This page lists recommended training resources for DevOps Team members. New team members should complete the **Onboarding Track** within their first 90 days. All team members should complete at least two courses per quarter to stay current with the Azure and DevOps toolchain.

## Recommended Learning Paths

### For Junior DevOps Engineers
1. Azure Fundamentals AZ-900 (Udemy)
2. Git: Getting Started (Pluralsight)
3. Azure DevOps: Getting Started (Pluralsight)
4. Azure DevOps Pipelines: Getting Started (Pluralsight)
5. Azure Monitor: Getting Started (Pluralsight)
6. KQL from Scratch (Pluralsight)
7. PowerShell: Getting Started (Pluralsight)

### For Mid-Level DevOps Engineers
1. Azure Administrator AZ-104 (Udemy)
2. Azure DevOps Engineer Expert AZ-400 (Udemy)
3. Bicep: Infrastructure as Code (Pluralsight)
4. Terraform on Azure (Udemy)
5. GitHub Actions: The Complete Guide (Udemy)
6. Docker and Kubernetes (Udemy)
7. Azure Key Vault (Pluralsight)

### For Senior DevOps Engineers / Leads
1. Azure Solutions Architect AZ-305 (Udemy)
2. Azure Security Technologies AZ-500 (Udemy)
3. Azure Kubernetes Service AKS (Pluralsight)
4. Site Reliability Engineering Fundamentals (Pluralsight)
5. DevSecOps Fundamentals (Pluralsight)
6. Chaos Engineering Fundamentals (Pluralsight)
