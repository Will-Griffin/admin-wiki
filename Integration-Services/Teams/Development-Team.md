# Development Team

Responsible for building and enhancing our Azure and BizTalk-based solutions.

Duties include:

- HIDEX API development

- Azure Functions, APIM, FHIR Proxy

- BizTalk legacy support & enhancements

- CI/CD enablement

- System modernization

- Code reviews & technical design
