## Welcome to the Integration Services Support Team!

This guide is designed to help new Support Team members get up to speed quickly. Complete each section in order during your first 90 days.

---

## Week 1 — Orientation & Access

### Day 1–2: Accounts & Access

Work with your manager to request access to the following systems:

| System | Purpose | Request Via |
|--------|---------|------------|
| HEAT Ticketing System | Incident & request tracking | IT Help Desk |
| NTST SF (Netsmart ServiceFirst) | Vendor ticket tracking | Manager |
| Azure Portal | Cloud monitoring & diagnostics | Azure DevOps admin / Manager |
| Azure DevOps | Work items, repos, pipelines | Manager |
| Microsoft Teams | Collaboration | IT Help Desk |
| BizTalk Admin Console | BizTalk monitoring | Manager |
| SharePoint / Wiki | Documentation | Manager |
| Email Distribution Lists | Team comms | Manager |

### Day 3–5: Tool Orientation

- [ ] Review this wiki starting with [Integration Services Overview](../../Integration-Services.md)
- [ ] Read the [Support Team SOPs](./Standard-Operating-Procedures-for-Support-Team.md)
- [ ] Shadow a senior analyst for ticket triage and resolution
- [ ] Observe a provider onboarding call
- [ ] Review the [HIDEX Troubleshooting Guide](../../Operations/Support/How-to-Troubleshoot-HIDEX-problem-by-using-Azure-Portal.md)
- [ ] Set up Azure Portal favorites: Log Analytics, Application Insights, APIM
- [ ] Install and configure Bruno and/or Postman

## Week 4 — Independent Work

- [ ] Handle your first HEAT ticket (with mentor available)
- [ ] Run a KQL query in Log Analytics to trace a message
- [ ] Execute a test case in Bruno/Postman against the QA environment
- [ ] Review a BizTalk suspended message with a senior analyst

## 30 / 60 / 90 Day Goals

| Milestone | Goals |
|-----------|-------|
| **30 Days** | Complete onboarding track training (see Training page). Independently handle P3/P4 tickets. Comfortable with Log Analytics KQL basics. |
| **60 Days** | Handle P2 tickets with minimal supervision. Assist with provider onboarding. Demonstrate proficiency with BizTalk monitoring. |
| **90 Days** | Independently manage full ticket lifecycle P1–P4. Lead a provider onboarding. Complete AZ-900 or ITIL 4 Foundation certification. Contribute to wiki documentation. |
