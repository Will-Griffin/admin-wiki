## Integration Services Support Analyst I

### Summary
Entry-level position responsible for daily operational monitoring, incident ticket handling, and basic troubleshooting of integration workflows. Works under the direction of senior analysts or the Support Lead.

### Key Responsibilities
- Monitor HEAT ticket queue and triage incoming requests per priority SLA.
- Perform basic troubleshooting of API and BizTalk integration issues using Azure Portal tools.
- Assist with vendor and provider onboarding documentation and coordination.
- Execute API test cases using Postman or Bruno and document results.
- Update and maintain wiki documentation and runbooks.
- Attend daily standups and change control meetings.

### Required Skills & Qualifications
- Bachelor's degree in Information Technology, Computer Science, or related field (or equivalent experience).
- Familiarity with ticketing systems (HEAT, ServiceNow, or similar).
- Basic understanding of REST APIs and HTTP request/response cycles.
- Basic SQL query knowledge.
- Strong written and verbal communication skills.
- Ability to work in a fast-paced, collaborative team environment.

### Preferred Skills
- Knowledge of EDI (X12) standards and healthcare data (HL7, FHIR).
- Experience with Azure Portal, Application Insights, or Log Analytics.
- ITIL Foundation certification or equivalent coursework.
- Familiarity with BizTalk Server monitoring.

### Tools Used
HEAT, Azure Portal, Log Analytics, Application Insights, Postman, Bruno, Azure DevOps, Microsoft Teams

## Integration Services Support Analyst II

### Summary
Mid-level position responsible for independently managing incidents, coordinating vendor onboarding, performing data validation and mapping reviews, and supporting change control processes.

### Key Responsibilities
- Independently triage, investigate, and resolve P1–P3 HEAT tickets within SLA.
- Lead provider onboarding calls and coordinate with Development and DevOps teams.
- Execute and document end-to-end API testing and validation.
- Write and maintain KQL queries for Log Analytics and Application Insights diagnostics.
- Perform BizTalk suspended message triage and document resolutions.
- Review and validate EDI transaction data against trading partner agreements.
- Assist with monthly provider cutoff procedures.
- Mentor Support Analyst I staff.

### Required Skills & Qualifications
- 2+ years of experience in IT support or integration support.
- Hands-on experience with Azure Portal, Log Analytics, and Application Insights.
- Proficiency in KQL (Kusto Query Language) for log investigation.
- Experience with REST API testing (Postman, Bruno).
- Understanding of EDI X12 transaction sets (837, 835, 270/271, etc.).
- Familiarity with HIPAA compliance requirements for healthcare data.
- Strong analytical and problem-solving skills.

### Preferred Skills
- Experience with BizTalk Server administration and monitoring.
- Knowledge of FHIR API standards and Azure APIM.
- ITIL 4 Foundation or higher certification.
- Microsoft Azure Fundamentals (AZ-900) certification.

### Tools Used
HEAT, NTST SF, Azure Portal, Log Analytics, Application Insights, Azure APIM, BizTalk Admin Console, Postman, Bruno, Azure DevOps, SQL Server

## Integration Services Support Lead / Senior Analyst

### Summary
Senior-level position responsible for leading the Support Team's daily operations, establishing and enforcing SOPs, managing escalations, coordinating with Development and DevOps leads, and driving continuous improvement in operational processes.

### Key Responsibilities
- Lead daily standups (8 AM and 2 PM) and facilitate team coordination.
- Manage P1 and P2 escalations and drive resolution.
- Own and maintain Support Team SOPs, runbooks, and wiki documentation.
- Coordinate Change Control submissions for the Support Team.
- Partner with Development and DevOps teams on incident root cause analysis.
- Oversee vendor and provider onboarding lifecycle.
- Manage team workload and capacity; assign tickets and tasks appropriately.
- Track KPIs (ticket volume, SLA compliance, MTTR) and report to management.
- Identify and implement process improvement opportunities.
- Interview and onboard new Support Team members.

### Required Skills & Qualifications
- 5+ years of IT support or integration operations experience.
- Deep expertise in Azure monitoring tools (Log Analytics, Application Insights, Azure Monitor).
- Strong working knowledge of EDI X12, HL7, and FHIR standards.
- Experience with BizTalk Server and Azure Integration Services.
- Proven experience managing SLA-driven operations and escalations.
- ITIL 4 Foundation certification required; ITIL Practitioner preferred.
- Microsoft Azure certifications (AZ-900 minimum; AZ-104 preferred).
- Leadership experience managing a small to mid-size technical team.

### Preferred Skills
- PMP or Agile/Scrum certification.
- Experience with healthcare interoperability (CMS regulations, HIPAA, HITECH).
- Familiarity with Azure DevOps boards and work item tracking.

### Tools Used
All tools used by the Support Team, plus: Azure DevOps (admin), Power BI (reporting), Microsoft Teams (coordination), Change Control system
