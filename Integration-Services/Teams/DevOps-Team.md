# DevOps Team

Responsible for automation, deployment pipelines, monitoring, observability, and operational excellence.

Duties include:

- Azure DevOps YAML pipelines

- GitHub Actions deployments

- Infrastructure-as-Code (Bicep/Terraform)

- Logging & monitoring (App Insights)

- Access provisioning

- Release packaging
